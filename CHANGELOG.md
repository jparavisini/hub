# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-20

### Added
- Initial public release of subnet protocol v1.0
- Zero-dependency Node.js static site generator
- `subnet.json` registry with JSON Schema validation
- Link tag handshake verification (`<link rel="subnet">`)
- Atom feed aggregation from member nodes
- OPML export for feed readers
- Embeddable Web Component widget with Shadow DOM
- Peer subnet discovery and metadata fetching
- GitHub Actions workflows for CI/CD
- Automatic daily rebuilds and GitHub Pages deployment
- Comprehensive documentation (README, CONTRIBUTING, CLAUDE.md)
- MPL-2.0 license for code, CC-BY-SA-4.0 for specification
